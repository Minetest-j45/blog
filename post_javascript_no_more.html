JavaScript No More - 3 April 2023X

Some of you will be happy to hear that this blog no longer replies on JavaScript to work. The way that things work now is that everytime I push to the repo, a GitHub Action is run which runs a Go script that I wrote.
This Go script generates the html pages for all the blog posts using the template that I have. It also remakes the index.html page to make sure the list of blog posts there is up to date. Now that the visitor's browser doesnt run JavaScript code to load the blog content, posts should load a bit faster, but you probably wont notice this unless you are on old hardware.
